Setting Up Your Environment: Installing Java and Eclipse

Before we begin writing code, it’s important to set up the tools needed to develop and run Java programs. This includes installing the Java Development Kit (JDK) and an Integrated Development Environment (IDE). In this guide, we’ll be using Eclipse, a widely-used and beginner-friendly IDE for Java development.

Installing the Java Development Kit (JDK)

The Java Development Kit (JDK) is a software package that provides the necessary tools to write, compile, and run Java programs. It includes the Java Runtime Environment (JRE), the Java compiler, and other essential components. If you want to learn more about JDKs read this article.

Note: If you already have a JDK installed, you can skip this step.

Steps to Install the JDK:

  1. Visit the official Oracle JDK download page here.
  2. Download the latest version of the JDK for your operating system (Windows, macOS, or Linux).
  3. Run the installer and follow the setup instructions. The default options are usually sufficient.
  4. After installation, verify the setup by opening a terminal or command prompt and typing: java -version
  5. If the JDK is installed correctly, you'll see the version number of the JDK in the output.

Installing Eclipse IDE

An Integrated Development Environment (IDE) provides a user-friendly interface to write, organize, and debug your code. Eclipse is a popular choice for Java because of its robust features and support for beginners. To learn more about IDEs read this article.

Steps to Install Eclipse:

  1. Go to the Eclipse Downloads page here.
  2. Click on the “Download” button under “Eclipse IDE for Java Developers.”
  3. Open the installer, select “Eclipse IDE for Java Developers”, and proceed with the installation.
  4. Once installed, launch Eclipse. You’ll be asked to select a workspace, which is a folder where your Java projects will be saved.

With both the JDK and Eclipse installed, your development environment is ready. In the next section, we’ll create your first Java program and walk through the basic structure of Java code.

Writing Your First Java Program: Basic Syntax and Structure

Now that your development environment is ready, it’s time to write your first Java program. This section will introduce you to the basic structure of a Java program and guide you through creating and running your first file using Eclipse.

Understanding Java Syntax and Structure

Java is a statically typed, object-oriented programming language. While we’ll explore those terms more deeply later, here’s a simple breakdown of what a basic Java program looks like:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}

Let’s break this down:

  • public class HelloWorld – Defines a class named HelloWorld. In Java, applications always start inside a class.
  • public static void main(String[] args) – The main method, which serves as the entry point for every Java program.
  • System.out.println("Hello, world!"); – This prints the text to the console; your program’s first output.

Creating and Running a Java File in Eclipse

Let’s walk through creating this simple program in Eclipse.

  1. Open Eclipse and select your workspace folder if prompted.
  2. Go to the top menu and select: File > New > Java Project.
  3. Name your project (e.g., FirstProgram) and click Finish.
  4. In the Project Explorer (usually on the left), right-click on the src folder: New > Class.
  5. In the dialog that appears:
    • Name your class HelloWorld.
    • Check the box that says "public static void main(String[] args)" to automatically generate the main method.
    • Click Finish.
  6. Eclipse will create a new .java file and open it in the editor. Inside the main method, type the following line: 
    System.out.println("Hello, world!");
  7. To run the program, right-click anywhere in the editor and select: Run As > Java Application.
  8. You should see the output "Hello, world!" appear in the Console panel at the bottom.

Congratulations! You've just written and executed your first Java program. This small step is the foundation of everything you'll build from here—apps, games, tools, and more. Variables and Data Types: Storing Information in Java

Now that you've written your first program, it’s time to start making your code more dynamic. Programs often need to store and work with data such as numbers, text, or even true/false values. In Java, we do this using variables.

What Is a Variable?

A variable is like a labeled container in your computer’s memory. You give it a name and store a value in it, so you can use that value later in your program.

Example:

int age = 25;

Here, we're creating a variable named age, giving it a value of 25, and specifying that it holds an integer (whole number) using the keyword int.

Common Java Data Types

Java is a statically-typed language, which means you must specify what type of data a variable will hold when you create it.

Java Variable Type Description Example
int Stores whole numbers without decimals. int count = 10;
double Decimal numbers. double price = 5.99;
char Single characters (letters). char grade = 'A';
boolean True or false values. boolean isActive = true;
String Sequences of characters (text). String name = "Luna";

Note: String is not a primitive type like the others—it's actually a class—but it's used often enough that it's treated like a basic type in early learning.

Declaring and Using Variables

Here's how you declare a variable and use it in a simple program:

public class VariablesExample {
    public static void main(String[] args) {
        String name = "Alex";
        int age = 21;
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
    }
}

When you run this program, you should see the following output:

Name: Alex
Age: 21

Note: You can choose any variable name you like, but it must start with a letter and can only contain letters, numbers, and underscores.

In this example:

  • We store a name and an age in variables.
  • We use the + operator to concatenate strings.
  • We use the System.out.println() method to print the output.

Understanding variables and data types is a crucial step in writing real programs. They allow you to create programs that react to and store different types of information, making your code much more powerful and flexible.
